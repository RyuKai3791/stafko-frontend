<h1 align="center">
STAFKO
</h1>
Stafko is a management app built in TypeScript using NextJS and a PostgreSQL Vercel database.
<br></br>

## How to start?
### Install dependencies:
```bash
cd ./app && npm install
```

<br></br>
App will run at: [http://localhost:3000](http://localhost:3000)
